# IntuneContentPrepTool
Gui for the MS Intune Content Prep Tool
Content of the MSI:
- MSCIPT-gui.exe (compiled ps1, please see sourcecode)
- IntuneWinAppUtil.exe (created by Microsoft, needed to create the .intunewim files)
